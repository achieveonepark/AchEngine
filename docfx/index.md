# AchEngine

Unity 프레임워크 툴킷 — VContainer DI, UI 시스템, 비동기 유틸리티, 구글 시트 연동 테이블 파이프라인을 포함합니다.

- [빠른 시작](articles/getting-started.md)

## API 레퍼런스

- [전체 네임스페이스 목록](api/index.md)

## 주요 모듈

| 모듈 | 설명 |
|------|------|
| `AchEngine` | AchTask, AchTimer — 비동기 유틸리티 |
| `AchEngine.UI` | UIView, UIService, UI 컴포넌트 |
| `AchEngine.DI` | ServiceLocator, AchEngineScope |
| `AchEngine.Managers` | AudioManager, SaveManager 등 |
| `AchEngine.Table` | 테이블 데이터 로드·조회 |
| `AchEngine.RedDot` | 알림 배지 시스템 |
| `AchEngine.Player` | 플레이어 데이터 컨테이너 |
