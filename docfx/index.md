# AchEngine API 레퍼런스

C# XML 주석에서 자동 생성된 클래스·메서드·프로퍼티 레퍼런스입니다.
가이드와 튜토리얼은 [AchEngine 문서](https://achieveonepark.github.io/AchEngine/)에서 확인할 수 있습니다 (ko / en / ja / zh).

## 네임스페이스

### Core — 비동기/타이머/디버그/네트워크
- <xref:AchEngine> — `AchTask`, `AchTimer`, `AchDebugConsole`, `HttpLink`, `RedDot` 등 루트 네임스페이스

### Dependency Injection
- <xref:AchEngine.DI> — `ServiceLocator`, `AchEngineScope`, `AchEngineInstaller`, `IServiceBuilder`

### UI System
- <xref:AchEngine.UI> — `UIView`, `UIService`, 컴포넌트 (`Draggable`, `AchButtonCooldown` 등)

### Managers
- <xref:AchEngine.Managers> — `AudioManager`, `AchSceneManager`, `PoolManager`, `ConfigManager`, `SaveManager`, `IAPManager`, `TimeManager`

### Data
- <xref:AchEngine.Table> — 타입 안전 테이블 로더
- <xref:AchEngine.Player> — 플레이어 데이터 컨테이너
- <xref:AchEngine.Save> — 세이브 추상화 레이어

### Gameplay
- <xref:AchEngine.Movement> — `AchMover`, `AchFollower`, `AchProjectile`
- <xref:AchEngine.Pathfinding> — A\* 그리드 길찾기

### Extensions
- <xref:AchEngine.Extensions> — `StringExt`, `ReflectionE`, `AchScriptableObject` 등 확장 모듈
