# AchEngine

**Unity 통합 개발 툴킷.** VContainer DI, UI System, Table Loader, Addressables, Localization 을 하나의 패키지로 제공합니다.

[**시작하기 →**](./guide/installation)

## 모듈

| 모듈 | 설명 |
|---|---|
| 🧩 **[DI (VContainer 래퍼)](./guide/di/)** | VContainer 를 직접 다루지 않고 `AchEngineInstaller` 와 `ServiceLocator` 로 서비스 등록·조회 |
| 🖼️ **[UI System](./guide/ui/)** | 레이어 기반 View 관리, 풀링, 전환 애니메이션 |
| 📊 **[Table Loader](./guide/table/)** | Google Sheets → C# 모델 자동 생성, 타입 안전 데이터 파이프라인 |
| 📦 **[Addressables](./guide/addressables/)** | 자동 그룹, 참조 카운트 캐싱, 씬 단위 수명 관리 |
| 🌐 **[Localization](./guide/localization/)** | JSON / CSV 다국어 워크플로우, 로케일 전환, 키 상수 생성 |
| ⚙️ **Project Settings 통합** | Project Settings &gt; AchEngine 에서 모든 모듈 설정·설치 상태 일괄 관리 |
