# AchEngine

**Unity 集成开发工具包。** 在一个包中提供 VContainer DI、UI System、Table Loader、Addressables 与 Localization。

[**开始使用 →**](./guide/installation)

## 模块

| 模块 | 说明 |
|---|---|
| 🧩 **[DI（VContainer 包装器）](./guide/di/)** | 无需直接使用 VContainer，通过 `AchEngineInstaller` 与 `ServiceLocator` 注册和解析服务 |
| 🖼️ **[UI 系统](./guide/ui/)** | 基于层级的视图管理、池化与过渡动画 |
| 📊 **[Table Loader](./guide/table/)** | 从 Google Sheets 自动生成 C# 模型，类型安全的数据流水线 |
| 📦 **[Addressables](./guide/addressables/)** | 自动分组、引用计数缓存、按场景管理生命周期 |
| 🌐 **[Localization](./guide/localization/)** | JSON / CSV 多语言工作流，区域切换，键常量生成 |
| ⚙️ **统一的 Project Settings** | 在 Project Settings &gt; AchEngine 中集中配置所有模块 |
