# AchEngine

**Unity 統合開発ツールキット。** VContainer DI、UI System、Table Loader、Addressables、Localization を 1 つのパッケージで提供します。

[**はじめる →**](./guide/installation)

## モジュール

| モジュール | 説明 |
|---|---|
| 🧩 **[DI (VContainer ラッパー)](./guide/di/)** | VContainer を直接扱わず `AchEngineInstaller` と `ServiceLocator` でサービスを登録・取得 |
| 🖼️ **[UI System](./guide/ui/)** | レイヤーベースの View 管理・プーリング・トランジション |
| 📊 **[Table Loader](./guide/table/)** | Google Sheets から C# モデルを自動生成、型安全なデータパイプライン |
| 📦 **[Addressables](./guide/addressables/)** | 自動グループ、参照カウントキャッシュ、シーン単位の寿命管理 |
| 🌐 **[Localization](./guide/localization/)** | JSON / CSV 多言語ワークフロー、ロケール切替、キー定数生成 |
| ⚙️ **Project Settings 統合** | Project Settings &gt; AchEngine で全モジュール設定を一括管理 |
