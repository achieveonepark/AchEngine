# AchEngine

**Unity development toolkit.** VContainer DI, UI System, Table Loader, Addressables, and Localization in a single package.

[**Get Started →**](./guide/installation)

## Modules

| Module | Description |
|---|---|
| 🧩 **[DI (VContainer Wrapper)](./guide/di/)** | Register and resolve services through `AchEngineInstaller` and `ServiceLocator` without working with VContainer directly. |
| 🖼️ **[UI System](./guide/ui/)** | Layer-based view management, pooling, and transitions. |
| 📊 **[Table Loader](./guide/table/)** | Auto-generate C# models from Google Sheets, type-safe data pipelines. |
| 📦 **[Addressables](./guide/addressables/)** | Auto groups, reference-counted caching, scene-scoped lifetime. |
| 🌐 **[Localization](./guide/localization/)** | JSON / CSV multilingual workflow, locale switching, key constants. |
| ⚙️ **Unified Project Settings** | Configure every module from Project Settings &gt; AchEngine. |
